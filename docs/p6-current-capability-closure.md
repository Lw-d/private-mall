## 自动回归覆盖

`pnpm smoke:transaction` 当前覆盖：

- API 健康检查。
- Swagger 用户订单分页文档。
- 小程序 mock 登录。
- 后台管理员登录。
- 后台更新积分抵扣规则。
- 小程序 public 积分规则同步。
- 后台创建 smoke 优惠券并上架。
- 小程序领取优惠券。
- 加入购物车和可用券查询。
- 创建待支付订单后锁券。
- 取消待支付订单后释放券。
- 使用优惠券和积分抵扣创建订单。
- 校验后台积分规则参与订单抵扣计算。
- 下单后扣减积分余额并写入积分消费流水。
- 积分抵扣断言完成后恢复原积分规则。
- mock 微信预支付和支付回调。
- 支付成功后核销优惠券。
- 后台搜索订单和发货。
- 用户确认收货和订单完成后积分发放。
- 后台经营概览优惠券和积分统计。
- 部分退款申请后按比例扣回订单发放积分。
- 部分退款成功后恢复订单履约状态并按比例返还抵扣积分。
- 剩余退款成功后订单变为已退款并结清积分扣回 / 返还。

最近一次完整回归：

```text
pnpm --filter @mall/server prisma:migrate:deploy
pnpm db:seed
pnpm smoke:transaction
```

结果：

```text
point redeem rule update ok: 80 points / 1 yuan
point redeem rule restored
Smoke transaction passed.
```
