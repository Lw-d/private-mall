## 外部环境依赖

以下事项无法在当前沙箱内完成，需要真实联调环境：

- 微信小程序 appId / secret。
- 微信支付商户号。
- 商户 API 证书序列号。
- 商户私钥文件。
- 微信支付 API v3 key。
- 微信支付平台公钥或平台证书。
- 公网 HTTPS API 域名。
- 微信商户平台回调地址配置。
- 微信开发者工具或真机调试。
- Docker / MySQL / Redis / API 可用环境。

## 当前环境限制

- 当前 Docker daemon 不可用，无法本地跑完整数据库链路。
- 当前 API 没有监听 `localhost:3000`，无法执行不跳过 API 的 `integration:check`。
- 当前沙箱内 `tsx` 会因 IPC pipe 被拦截，`smoke:transaction` 和 `integration:check` 需要在沙箱外运行。
- 当前 macOS 沙箱下 Taro build 有已知 panic / 挂起风险，小程序构建建议放到非沙箱本机或 CI。
